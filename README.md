# A Basic Swift Env on Linux

## My Setup:

* ThinkPad X250 (x86_64)
* Ubuntu 16.04
* NVim

## Install
```bash
./install.bash
```

## Test

New console:
```
swift --version
# should see:
# Swift version 3.0 (swift-3.0-PREVIEW-1)
# Target: x86_64-unknown-linux-gnu
```
